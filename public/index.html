SELF AS SYSTEM

This is an interactive Data Portrait website related to Instagram data. It reflects multiple sides of one person that can be represented through the information from the Instagram account. Developed from some small sketches. The portrait mainly includes four parts, the data loop, interactive like system, slit-scan camera, and saved images’ file path. Move around MOUSE to interact with the like system. Use the CAPTURE button or press ‘c’ key on the keyboard to initiate a slit-scan camera. And use the SAVE button or press ‘s’ key to save the image with a random 36 digit JPG file just like the file names within the Instagram data.
